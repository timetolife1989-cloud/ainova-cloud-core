# I18N — Nemzetköziesítés feladatok

## Állapot

A rendszer támogatja: **HU / EN / DE**
- Server-side: `getTranslationsForLocale()` → DB + fallback JSON
- Client-side: `useTranslation()` hook → `I18nProvider` context

## Hol működik

- ✅ Dashboard fejléc (Vezérlőpult / Dashboard / Kontrollzentrum)
- ✅ Admin panel menü nevek
- ✅ Login oldal fő elemek (Username, Password, Sign in)
- ✅ Nyelvváltó HU/EN/DE (Header dropdown)

## Hol NEM működik (hardcoded magyar stringek)

### Login / Auth
- ⬜ `app/api/auth/login/route.ts` — Zod validációs üzenetek ("A felhasználónév nem lehet üres")
- ⬜ `app/api/auth/login/route.ts` — Hiba válasz ("Váratlan hiba történt.")
- ⬜ `app/api/setup/route.ts` — Setup wizard hibaüzenetek

### Modul UI-k
- ⬜ `modules/workforce/` — Form labelek, oszlopnév fejlécek
- ⬜ `modules/tracking/` — Státusz nevek (Nyitott, Folyamatban, stb.)
- ⬜ `modules/fleet/` — Űrlap feliratok
- ⬜ Minden modul: Táblázat fejlécek, gomb szövegek, placeholder-ek

### Admin Panel
- ⬜ `components/admin/` — Kártya leírások
- ⬜ Dialógus ablakok, megerősítő üzenetek

## Tennivaló sorrend

1. Login/auth hibaüzenetek → i18n kulcsok (legkritikusabb — a user ezt látja)
2. Modul UI form labelek → useTranslation()
3. Admin panel szövegek → useTranslation()
4. Toast üzenetek → i18n kulcsok
